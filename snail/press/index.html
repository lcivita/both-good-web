Overview

Don’t Touch the Snail is a desktop overlay game where a snail slowly crawls toward your mouse cursor. If it ever reaches you, the game ends permanently.

Description

Don’t Touch the Snail is an experimental desktop overlay game built around a deliberately simple and ridiculous premise: a snail is always making its way toward the player’s cursor.

Whenever the game is running, the snail slowly advances. There are no resets, retries, or difficulty settings. If the snail ever reaches the cursor, the player's score is locked on the leaderboard and the game ends permanently.

In that sense, owning the game is part of the joke. Don’t Touch the Snail asks players to knowingly take on a problem that is trivial, harmless, and impossible to fully escape, turning a minimal mechanic into a long-term, self-imposed challenge that lives quietly, but inconveniently, on their desktop.
